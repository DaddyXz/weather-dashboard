# Weather Dashboard

## Project Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/DaddyXz/weather-dashboard.git
   cd weather-dashboard
   ```

2. **Install dependencies**:
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

## Features

- **Current Weather**: Displays the current weather data of a selected city.
- **Five-Day Forecast**: Provides a detailed forecast of the next five days.
- **Responsive Design**: Mobile-friendly interface to ensure usability across devices.
- **Search Functionality**: Users can search for any city to retrieve weather data.
- **Real-Time Updates**: Fetches real-time data using weather APIs.

## Requirements
- Node.js v14 or higher
- npm v6 or higher
